macOS 支持 Unicode 输入，因此你可以通过 Unicode 来直接插入拼音音标。你可以使用以下快捷键来输入常见的拼音音标：

- `Option + u`：输入 **ˉ**（第一声）
- `Option + e`：输入 **ˊ**（第二声）
- `Option + i`：输入 **ˇ**（第三声）
- `Option + `（反引号）：输入 **ˋ**（第四声）
- `Option + .`（点）：输入 **˙**（轻声）
