抽 ​

（chōu）：

➀ 拔出，抽出。《左传·昭公二十一年》：“～矢，城射之。”（城：人名。）李白《宣州谢朓（tiǎo）楼饯别校书叔云》诗：“～刀断水水更流。”引提取。元稹《织妇词》：“今年私税～征早。”引植物出芽。束皙《补亡》诗：“草以春～。”（草在春天发芽。）

➁ 拔掉。《诗经·小雅·楚茨（cí）》：“楚楚者茨，言～其棘。”（楚楚：茂盛。茨：蒺藜（jí lí）言：动词词头。棘：带刺灌木。）引提拔。《南史·颜延之传》：“延之昔坐事屏斥，复蒙～进。”

➂ 抒发。屈原《九章·抽思》：“与美人～怨兮。”

怞 ​

（chōu）：

➀ 同“妯”。不平静。《诗经·小雅·鼓钟》：“忧心且妯。”《说文》引作“忧心且怞”。

➁ yóu（由）。[怞怞]忧愁的样子。王褒《九怀·危俊》：“永余思兮～～。”

特别说明：古汉语字典读音标注为chōu，百度汉语则为chóu。

瘳 ​

（chōu）：

➀ 病好了。《诗经·郑风·风雨》：“既见君子，云胡不～！”（胡：何。）《后汉书·华佗传》：“病皆～。”引治，治愈。《庄子·列御寇》：“国其有～乎？”欧阳修《汝瘿（yǐng）答仲仪》：“政可～民瘼（mò）。”（瘼：疾苦。）

➁ 损失，损害。《国语·晋语二》：“君不度而贺大国之袭，于己也何～？”

仇 ​

（chóu）：

➀ qiú（求）。同伴。《诗经·周南·兔罝（jū）》：“赳赳武夫，公侯好～。”又配偶。曹植《浮萍篇》：“来为君子～。”

➁ 仇敌。《史记·秦始皇本纪》：“属疏远，相攻击如～雠（chóu）。”（属：亲属。：仇敌。）成语有“同仇敌忾”。又仇恨。《史记·郭解传》：“雒（luò）阳人有相～者。”

➂ qiú（求）。姓。

䌷 ​

（chóu）：

➀ 丝织品。《盐铁论·散不足》：“茧～缣练者，婚姻之嘉饰也。”这个意义后来写作“绸”。

➁ chōu（抽）。缀集。《史记·太史公自序》：“（父）卒三岁而迁为太史令，～史记石室金匮之书。”

➂ chōu。抽引。宋玉《高唐赋》“～大弦而雅声流，冽风过而增悲哀。”[䌷绎]引出头绪。《汉书·谷永传》：“燕见～～，以求咎愆（qiān ）。”（燕见：指在皇帝内廷朝见。）

俦 ​

（chóu儔‌）：

伴侣。李白《赠崔郎中宗之》诗：“草木为我～。”又同类，类别。《三国志·魏书·崔林传》：“忠直不回，则史鱼之～。”（回：邪恶。史鱼：人名。）引相比。杜甫《陪王侍御同登东山最高顶宴姚通泉》诗：“姚公美政谁与～？”【注意】汉代以前，“俦”一般都写作“畴”。如《荀子·劝学》：“草木畴生。”（同类的草木生长在一起。）

畴 ​

（chóu疇）：

➀ 已耕作的田地。《荀子·富国》：“其田～秽。”（秽：荒芜。）泛田亩。《吕氏春秋·慎大》：“朝不易位，农不去～。”用作动词，培土。《淮南子·俶真》：“～以肥壤。”

➁ 谁。《列子·天瑞》：“运转亡已，天地密移，～觉之哉？”

➂ 同类，类别。《荀子·劝学》：“草木～生，禽兽群焉，物各从其类也。”《战国策·齐策三》：“物各有～。”这个意义后又写作“俦”。

➃ [畴人]历算家。《史记·历书》：“～～子弟分散。”清代阮元著有《畴人传》。

➄ [畴昔]过去，以前。《左传·宣公二年》：“～～之羊，子为政，今日之事，我为政。”

踌 ​

（chóu躊）：

[踌躇]1.徘徊，犹豫。曹丕《出妇赋》：“马～～而回顾，野鸟翩（piān）而高飞。”（翩：急飞。）向秀《思旧赋》：“心徘徊以～～。”（以：而。）这个意义又写作“踌蹰”、“踌伫”。2.从容自得的样子。《庄子·养生主》：“提刀而立，为之四顾，为之～～满志。”

筹 ​

（chóu籌）：

➀ 计数的用具。《汉书·五行志下之上》：“～所以纪数。”今有熟语“筹码”。

➁ 谋划，计划。《盐铁论·非鞅》：“夫蓄积～策，国家之所以强也。”（积蓄物资财富，善于计划，是国家所以富强的原因。）又计策，计谋。《晋书·宣帝纪》：“非经国远～。”（经国：治国。）成语有“一筹莫展”。

惆 ​

（chóu）：

[惆怅][惆然]伤感，失意。陶潜《归去来兮辞》：“奚惆怅而独悲？”（奚：为什么。）《荀子·礼论》：“则其于志意之情者惆然不嗛（qiè）。”（嗛：满足。）

注：古汉语字典中嗛备注为qiè，实际百度嗛qiǎn。

绸 ​

（chóu）：

➀ 缠绕。《尔雅·释天》：“素锦～杠。”（素：白。杠：旗杆。）[绸缪]1.缠绕。《诗经·唐风·绸缪》：“～～束薪。”（束薪：成捆的柴。）成语有“未雨绸缪”（喻事先做好准备。）2.情意缠绵。《三国志·蜀书·先主传》：“先主至京见（孙）权，～～恩纪。”又如“情意绸缪”。

➁ 一种丝织品。《周书·武帝纪》：“庶人已上，唯听衣～，绵～，丝布、圆绫、纱、绢、绡、葛、布等九种。”

➂ 通“稠”。多而密。《北史·北海王详传》：“往来～密。”

稠 ​

（chóu）：

多而密。《三国志·蜀书·关羽传》：“先主与二人寝则同床，恩若兄弟，而～人广坐，侍立终日。”（二人：指关羽、张飞。）成语有“地窄人稠”，双音词有“稠密”。引浓厚，与稀相对。贾思勰《齐民要术·种谷》：“挠令洞洞如～粥。”（搅拌使它像浓粥一样。洞洞：很浓的样子。）

裯 ​

（chóu）：

单被。《诗经·召南·小星》：“肃肃宵征，抱衾与～。”泛被子。杨万里《霜夜无睡闻画角孤雁》诗：“拥～起坐何人伴？只有残灯半晕青。”

酬 ​

（chóu酧、醻）：

➀ 客人给主人祝酒后，主人再次给客人敬酒。《世说新语·赏誉下》：“温元甫、刘王乔、裴叔则俱至，～酢终日。”（酢：客人向主人回敬酒。）杜牧《念昔游》诗：“樽（zūn）前自献自为～。”（樽：酒器。献：主人给客人敬酒。）

➁ 酬报，报答。李白《走笔赠独孤驸马》诗：“壮心剖出～知己。”引应对，赠答。刘知几《史通·惑经》：“其间切磋～对。”（切磋：相互商讨研究。）谢灵运《拟魏太子邺中集诗·应王玚（yáng）》：“调笑辄（zhé）～答。”（辄：就，即。）

➂ 偿付，偿还。《新唐书·李朝隐传》：“成安公主夺民园，不～直。”（直：价钱。）

➃ 实现（志愿）。李频《春日思归》诗：“壮志未～三尺剑，故乡空隔万重山。”

愁 ​

（chóu）：

忧虑，发愁。《左传·襄公二十九年》：“哀而不～，乐而不荒。”李白《秋浦歌·十五》：“白发三千丈，缘～似个长。”（缘：因为。个：这般。）又形容凄惨、惨淡的景象。谢惠连《雪赋》：“寒风积，～云繁。”辛弃疾《鹧鸪天·赋牡丹》：“～红惨绿今宵看，恰似吴宫教阵图。”

雠 ​

（chóu讐）：

➀ 应答。《诗经·大雅·抑》：“无言不～，无德不报。”引相应，应验。《汉书·窦田灌韩传》：“于是上使御史簿责婴所言灌夫颇不～。”（簿：文书。婴、灌夫：人名。）《史记·封禅书》：“其方尽，多不～。”（方：方术。尽：用完了。）

➁ 相当，相匹配。《汉书·霍光传》：“卒不得遂其谋，皆～有功。”（雠有功：指其功相等。）

➂ 售，卖出去。《墨子·经下》：“贾宜则～。”（贾：价。）《史记·高祖本纪》：“高祖每酤（gū）留饮，酒～数倍。”

➃ 仇敌，仇人。《尚书·微子》：“小民方兴，相为敌～。”又仇恨。屈原《九章·惜诵》：“又众兆之所～。”（众兆：众人。）

➄ [雠校]校对，校勘。《后汉书·和熹邓皇后纪》：“乃博选诸儒刘珍等……诣东观～～传记。”（东观：汉代皇帝藏书的地方。）

幬 ​

（chóu）：

➀ 床帐。宋玉《招魂》：“蒻（ruò）阿拂壁，罗～张些。”（蒻：细而软。阿：细缯。些：语气词。）泛帷幕。韩缜《东山寺》诗：“像设严珠殿，经声隐绛～。”

➁ 车帷。《史记·礼书》：“大路之素～也。”（路：车。）

➂ dào（道）。覆盖。《左传·襄公二十九年》：“如天之无不～也，如地之无不载也。”

丑1 ​

地支第二位。丑属牛。又十二时辰之一，等于现在的凌晨一时至三时。见“干”字。

丑2 ​

（chǒu醜）：

➀ 相貌难看，与“美”相对。《史记·西门豹传》：“呼河伯妇来，视其好～。”（河伯：指河神。好：美。）

➁ 恶，不好。《诗经·小雅·十月之交》：“日有食之，亦孔之～。”《汉书·项籍传》：“今尽王故王于～地。”引羞耻，耻辱。《庄子·外物》：“终身之～。”又憎恶。《荀子·荣辱》：“我甚～之。”

➂ 类。《国语·楚语下》：“官有十～。”《尔雅·释鸟》：“凫（fú），雁～。”（凫：野鸭。）引类似。《孟子·公孙丑下》：“今天下地～德齐，莫能相尚。”【注意】在古代，“丑”和“醜”是两个字，意义各不相同。上述义项都不写作“丑”。现在“醜”简化为“丑”。

臭 ​

（chòu）：

➀ xiù（嗅）。气味。《荀子·王霸》：“口欲綦（qí）味，鼻欲綦～。”（綦：极，尽。味：滋味。）

➁ xiù（嗅）。嗅，闻。《荀子·荣辱》：“彼～之而无嗛（qiè）于鼻，尝之而甘于口。”（嗛：满足。）这个意义后来写作“嗅”、“齅”。

➂ 臭，与“香”相对。仲长统《昌言·理乱》：“三牲之肉，～而不可食。”（三牲：指牛、羊、猪。）

注：古汉语字典备注嗛读作qiè，百度汉语查询只有xián和qiǎn的读音。
