## 仓

（cāng 倉）：

➀ 粮仓。《商君书·去强》：“～府两虚，国弱。”

➁ 船舱。杨万里《初二日苦热》诗：“船～周围各五尺。”这个意义后来写作“舱”。

➂ 通“苍”。青色。《礼记·月令》：“驾～龙。”（龙：指大马。）

## 伧

（cāng 傖）：

粗野，粗俗。《陈书·周铁虎传》：“周铁虎，不知何许人也。梁世南渡，语音～重，膂（lǚ）力过人。”[伧父（fǔ）]粗野鄙贱的人。《晋书·左思传》：“此间有～父，欲作《三都赋》。”
