## 黪

（cǎn）：

灰黑色。引申为色彩暗淡。沈括《梦溪笔谈》卷二：“以～衣蒙之，谓之凉衫。”[黪黩]混浊不清。杜甫《三川观水涨二十韵》：“何时通舟车，阴气不～～。”

## 璨

（càn）：

明亮。《旧唐书·柳宗元传》：“～若珠贝。”[璨璨]明亮的样子。白居易《黑龙饮渭赋》：“气默默以黯黯，光～～而烂烂。”
