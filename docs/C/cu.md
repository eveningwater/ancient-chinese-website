## 粗

（readcūread麤）：

➀ 粗米，粗粮。《左传·哀公十三年》：“粱则无矣，～则有之。”（粱：精美的饭食。）o引o粗糙，粗略。《荀子·正名》：“～布之衣。”《荀子·正名》：“故愚者之言，芴（hū）然而～。”（芴然：没有根据的样子。）

➁ 粗大。《礼记·月令》：“其器高以～。”（以：而。）

➂ 大概，大致。诸葛亮《谕谏》：“纲纪～定。”（纲纪：法规。）

➃ 鲁莽，粗鲁。《三国志·吴书·吕蒙传》：“甘宁～暴好杀。”（甘宁：人名。）
